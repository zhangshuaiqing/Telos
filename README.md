# Telos

> **τέλος** — 从感知到行动的完整闭环 Agent

## 是什么

Telos 是一个低算力、API-first 的机器人认知系统。它不依赖本地 GPU，通过多模态感知（视觉、语音、本体感）理解世界，由云端 LLM 推理决策，输出到 7 种执行器原语驱动硬件，形成完整的闭环。

```
世界 → 感知 → 认知(LLM) → 决策 → 执行(7原语) → 世界
```

## 为什么

- **无 GPU 也能跑** — 端侧只做轻量采集和规则安全，重推理走 API
- **可扩展** — 新感知通道、新执行器只需实现统一接口
- **完整闭环** — 不只是聊天，是真正能驱动机器人的系统
- **安全分层** — 云端建议 → 端侧校验 → STM32 硬件急停

## 快速开始

```bash
# 安装
pip install -e .

# 设置 API Key
export DEEPSEEK_API_KEY="your-key"

# 空跑测试 (不调用 API)
python main.py --dry-run

# 运行 Agent
python main.py --task "向前走 2 米"
```

## 架构

```
          ┌─────────────────────────┐
          │     感知层 (Perception)   │
          │  视觉│语音│本体感│可扩展  │
          └───────────┬─────────────┘
                      ▼
          ┌─────────────────────────┐
          │     认知引擎 (Cognition) │
          │   云端 DeepSeek/Kimi API │
          └───────────┬─────────────┘
                      ▼
          ┌─────────────────────────┐
          │     执行层 (7 原语)      │
          │  旋转速度│旋转位置│直线   │
          │  开关│泵│夹爪│能量束     │
          └─────────────────────────┘
```
